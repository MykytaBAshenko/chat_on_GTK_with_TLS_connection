# chat_on_GTK_with_TLS_connection
Simple chat on C with GTK3 and TLS connection
### Installation:
>**1. brew install gtk+3**  
>**2. git clone**  
>**3. make**  
>**4. uchat_server 1337 0.0.0.0 admin**  
>**5. uchat ip(127.0.0.1) 1337**  
### Technologies used:
>**1. GTK+3**  
>**2. JSON**  
>**3. CSS**  
>**4. TLS**  
>**5. SQLite3**
